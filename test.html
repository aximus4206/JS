1. Итератор это объект имеющий метод "некст"(возвращаяющий следующее значение, если таково имееться), способный перебрать нужный нам объект. Ниже пример получения встроенного итератора путем вызова его напрямую, например у строки для побуквенного вывода ее содержимого: let someStr = "somestringforexample"; let iterator = someStr[Symbol.iterator](); while(true) { let result = iterator.next(); if (result.done) break; console.log(result.value); } С массивами так же.

2. Итерируемым объект являеться в том случае, если он имеет метод Symbol.iterator. Итерируемой по умолчанию являеться например строка или массив, но строка например являеться и итерируемым объектом и псевдомассивом П.С."поэтому не знаю подходящий ли это пример".
